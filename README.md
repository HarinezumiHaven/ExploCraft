# About Mod
## Mod items:
### - Explosive Barrel
- Parameters can be edited with amount of gunpowder and redstone
	- REDSTONE - amount of time before explosion
		- 0 - 5s
		- 1-3 - 30s
		- 4-8 - 60s
		- 9-12 - 90s
		- 13+ - 120s
	- GUNPOWDER - explosion power
		- r ~= gunpowder/2 [blocks]
		- MAX ~= 10 [blocks]
- Can be activated with redstone signal
- If opening after activation:
	- 50% - instant explosion
	- 50% - possibility to grab items from barrel and prevent explosion
